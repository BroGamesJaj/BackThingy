
## Description

Nestjs backend for Thingy music player.

## Project setup
1. download packages
```bash
$ npm install
```
2. create a .env based on the .env.example

## Compile and run the project

```bash
# development
$ npm run start:debug

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Endpoints

The full documentation of the endpoints can be viewed under /api after running the project

## Run tests

```bash
$ npm run test
```
